# Unsupervised Learning: K-Means & PCA Analysis

## Project Overview
This project explores two fundamental techniques in unsupervised machine learning: **K-Means Clustering** and **Principal Component Analysis (PCA)**. 

The focus is on understanding how to group unlabeled data and how to reduce the dimensionality of complex datasets while preserving the maximum possible variance.

## Key Components
1. **K-Means Clustering:**
   - Implementation of the algorithm from scratch/using Scikit-Learn.
   - Use of the **Elbow Method** to determine the optimal number of clusters ($k$).
   - Analysis of cluster centroids and intra-cluster variance.
2. **Principal Component Analysis (PCA):**
   - Dimensionality reduction to visualize high-dimensional data.
   - Analysis of **Explained Variance Ratio** to identify the most significant features.
   - Data standardization and covariance matrix computation.

## Tools Used
- **Python**: Core programming.
- **NumPy & Pandas**: Data manipulation.
- **Matplotlib/Seaborn**: Cluster visualization and Scree plots.
